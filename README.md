# Forecast Redelong

Sistem prakiraan hujan berbasis ensemble untuk dukungan monitoring PLTA Redelong.

## Isi sistem

- Forecast hujan multi-source
- Titik PLTA Redelong dan GPM catchment
- Peta hujan interaktif
- Dashboard hujan per titik
- Output CSV dan GeoJSON untuk verifikasi

## Titik lokasi

- PLTA Redelong
- GPM1
- GPM2
- GPM3
- GPM4
- GPM5
- GPM6
- GPM Grid TamaTue

## Catatan

Sistem ini adalah prototype monitoring awal. Hasil forecast tetap perlu divalidasi dengan observasi aktual sebelum digunakan untuk keputusan operasional kritikal.
